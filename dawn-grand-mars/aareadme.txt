A description of the bundle and how to get started is provided in the bundle description document.

If you have access to the bundle, it is located in the document directory, with a file name of "Dawn_GRaND_Mars_bundle_description.pdf".

Otherwise, you can find it by using the PDS4 LID of "urn:nasa:pds:dawn-grand-mars:document:dawn_grand_mars_bundle_description.pdf".

An overview of the entire GRaND archive can be found in the GRaND ancillary bundle document directory, named "Dawn_GRaND_archive_description.pdf", with a PDS4 LID of "urn:nasa:pds:dawn-grand-ancillary:document:dawn_grand_archive_description.pdf"
